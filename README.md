## About

ZSRX is an interactive educational and scientific software platform for exploring synchrotron radiation through mathematical derivations, interactive visualizations, and publication-quality graphics.

Current features include:

- Interactive Plotly visualizations
- MathJax-rendered equations
- Scientific derivations
- Interactive parameter exploration
- Publication-quality figures
