# ZSRX

## Zalloum Synchrotron Radiation Explorer

**Version 1.0.0**  
**Part of the Zalloum Physics Suite**

ZSRX is an interactive scientific visualization platform for exploring synchrotron radiation physics through mathematical derivations, real-time parameter controls, and publication-quality graphics.

---

## Developed by

**Prof. Othman H. Y. Zalloum**  
Palestine Polytechnic University

---

## Features

- Interactive HTML dashboard
- Plotly scientific visualizations
- MathJax-rendered equations
- Collapsible derivation panels
- Real-time sliders for electron energy, magnetic field, bending radius, observation angle, and frequency
- Dynamic plots for critical frequency, spectral power, total radiated power, and relativistic beaming
- Dark publication-style scientific interface

---

## Launch

Open the dashboard directly in a browser:

```text
app/ZSRX_v1.0.html
```

No installation is required. The dashboard uses Plotly.js and MathJax through CDN links.
